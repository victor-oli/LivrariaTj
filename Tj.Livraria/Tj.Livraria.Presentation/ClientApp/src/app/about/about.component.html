Esta App foi feita como forma de desafio
          para testar algunas das minhas habilidades,
    espero que gostem. =)

Fiz alguns pequenos testes usando TDD, Optei por arquitetura DDD para por em ação as boas práticas de OOP, usei o SqlServer para persistir os dados mas poderia ter usado várias coisas legais como MongoDb e Redis para otimizar a performace. No front-end um pouco de Angular com Bootstrap.

OBS: Se você rodar a App na sua máquina, não esqueça de ajustar a connectionstring no arquivo de configurações!
